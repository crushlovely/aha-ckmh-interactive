# CKM Connection Explorer — v1 Plan

**Project:** Interactive Learning Experience for CKM Education
**Client:** American Heart Association — CKMH Initiative
**Prepared by:** Crush & Lovely
**Target Launch:** Late May / Early June 2026
**Build Target:** `src/components/ckmh-connections.vue`

---

## 1. Project Vision

Transform a static informational page into an interactive Venn diagram experience that teaches consumers how cardiac, kidney, and metabolic health are interconnected — without requiring any prior medical knowledge, any account creation, or any clinical language.

The current Vue component renders four text sections in a card grid. The redesign replaces this with a patient-led, discovery-based Venn diagram where users click, hover, and explore to reveal layered content about each health domain and, critically, the overlaps between them.

---

## 2. UX Strategy

### 2.1 Core Interaction Model

The centerpiece is a **three-circle Venn diagram** representing Heart, Kidney, and Metabolic health. The diagram creates **seven clickable regions**:

| Region | Label | Content Focus |
|--------|-------|---------------|
| H | Heart | Cardiac health basics — what the heart does, what damages it |
| K | Kidney | Kidney health basics — filtering, blood pressure, fluid balance |
| M | Metabolic | Metabolic health basics — blood sugar, weight, energy processing |
| H∩K | Heart + Kidney | How heart and kidney disease accelerate each other |
| H∩M | Heart + Metabolic | How diabetes/obesity damage blood vessels and the heart |
| K∩M | Kidney + Metabolic | How diabetes and high BP are leading causes of kidney disease |
| H∩K∩M | All Three | The full CKM connection — integrated risk, integrated action |

### 2.2 User Flow

```
Landing (Hero)
  │
  ▼
Venn Diagram — default "resting" state
  │  User hovers/taps a region
  ▼
Region Highlight — short tooltip preview (one sentence)
  │  User clicks/taps
  ▼
Content Panel — slides in from right (desktop) or bottom (mobile)
  │  Contains: plain-language explanation, "Why this matters to you"
  │  callout, key stats, and a call-to-action
  │  User can navigate between regions from within panel
  ▼
Completion Nudge — after visiting 4+ regions, a gentle prompt
  encourages the user to explore the center overlap
  │
  ▼
Final CTA — "Talk to your doctor about CKM screening"
  with link to heart.org/myCKMhealth
```

### 2.3 Key UX Principles

**Patient-led, not organization-led.** There is no forced path. Users choose which circle to explore first. The diagram invites curiosity rather than prescribing a curriculum.

**Zero-baseline assumption.** Every piece of content is written for someone who has never heard the term "CKM syndrome." Medical terms are always accompanied by a plain-language equivalent in parentheses (e.g., "blood glucose (blood sugar)").

**Repeatable.** The experience should feel worthwhile on a second visit. Session-based cookies can track which regions a user has previously explored, gently highlighting unexplored areas on return without gating any content.

**No account required.** All content is accessible immediately. No login walls, no email gates. Anonymous analytics only.

**Mobile-first.** The Venn diagram must work as a tap interaction on small screens. The three-circle layout becomes vertically stackable or horizontally scrollable at narrow viewports, with overlap regions accessible through explicit tap targets.

## 3. UI Design Direction

### 3.1 Visual Language

**Delightful but serious.** The AHA brand uses a confident red (#C10E21) as its primary accent. The interactive should feel warm and modern — not sterile or clinical.

**Palette:**

| Role | Color | Usage |
|------|-------|-------|
| AHA Red | `#C10E21` | Hero background, primary CTAs, headings |
| Heart | Warm red / rose | `#E05A6D` — Heart circle fill |
| Kidney | Teal / blue-green | `#3A9BA5` — Kidney circle fill |
| Metabolic | Amber / gold | `#D4943A` — Metabolic circle fill |
| Overlaps | Blended tones | Derived from parent circles at reduced opacity |
| Center | Deep plum / neutral | `#5C3D5E` — Where all three meet |
| Background | Off-white | `#F7F5F2` — Warm, approachable |
| Text | Dark charcoal | `#2D2D2D` — High contrast, accessible |

**Typography:** Inherit from AHA/heart.org base styles. Headings bold and large for scanability. Body text at 16–18px for readability. Grade 5–8 reading level throughout.

### 3.2 Venn Diagram Design

The diagram uses three overlapping circles with semi-transparent fills so overlaps create natural color blends. Each circle has a clear label at its center. Overlap regions show a small icon or "+" indicator to signal they are interactive.

**States:**

| State | Visual Treatment |
|-------|-----------------|
| Default | Soft fills, subtle drop shadows, labels visible |
| Hover | Hovered region brightens, slight scale pulse, cursor changes |
| Active/Selected | Region fully opaque, other regions dim to 40% opacity |
| Visited | Small checkmark badge on the region label |
| Unexplored nudge | Gentle pulsing glow on unvisited regions (after 3+ visits) |

### 3.3 Content Panel

When a region is selected, a panel slides into view containing the educational content. On desktop this appears to the right of the diagram; on mobile it slides up as a bottom sheet.

**Panel anatomy:**

```
┌─────────────────────────────────────┐
│  ✕ Close                            │
│                                     │
│  [Icon]  HEART + KIDNEY             │
│                                     │
│  Heading (plain language)           │
│  ─────────────────────              │
│  Body text (2–3 short paragraphs)   │
│                                     │
│  ┌─ WHY THIS MATTERS TO YOU ──────┐ │
│  │  Empowering callout with a     │ │
│  │  personal, motivating tone     │ │
│  └────────────────────────────────┘ │
│                                     │
│  KEY NUMBERS                        │
│  • 1 in 3 adults have high         │
│    cholesterol                      │
│  • Nearly half have high BP         │
│                                     │
│  [→ Learn more at heart.org]        │
│                                     │
│  ← Prev Region    Next Region →    │
└─────────────────────────────────────┘
```

### 3.4 Responsive Behavior

**Desktop (≥ 960px):** Venn diagram on the left (~55% width), content panel on the right (~45% width). Panel slides in/out with a smooth transition.

**Tablet (600–959px):** Venn diagram full width above, content panel slides up from bottom as an overlay (60% viewport height).

**Mobile (< 600px):** Venn diagram simplified — circles are slightly smaller with larger tap targets on overlap zones. Content panel is a full-screen bottom sheet with swipe-to-dismiss.

## 4. Content Strategy

### 4.1 Voice and Tone

All copy follows the AHA-approved consumer messaging guidelines:

- **Accessible and educational** — never clinical or condescending
- **Grade 5–8 reading level** — use bulleted lists to lower reading complexity
- **Empowering, not frightening** — focus on what people *can do*, not just what can go wrong
- **"You" language** — speak directly to the reader as an individual

### 4.2 Content per Region

Each of the seven Venn regions follows a consistent content template:

**A. Headline** (5–8 words, plain language)
Example: "When your heart and kidneys team up"

**B. Body** (2–3 short paragraphs, ~80–120 words total)
Explains the health domain or connection in everyday language. Uses the team/body-as-connected-system metaphor from AHA messaging: "Your heart, kidneys, and the way your body creates energy have different jobs, but they share the work to keep you healthy."

**C. "Why This Matters to You" Callout** (~30–50 words)
A single motivating statement. Example: "If you're already working on one part of your health — like managing blood pressure — the good news is other parts benefit too."

**D. Key Numbers** (2–3 stats from AHA-approved data)
Drawn from the approved messaging documents. Examples:
- "Nearly half of U.S. adults have high blood pressure."
- "Only about 10% of U.S. adults have excellent CKM health."
- "Nearly 9 out of 10 adults age 20+ have at least one CKM risk factor."

**E. Call to Action**
Links to heart.org/myCKMhealth or relevant AHA resource. Tone: "Learn more about steps you can take" — not pushy, not urgent.

### 4.3 Content Map — Draft Outlines

**Heart (H)**
- Headline: "Your heart keeps everything flowing"
- Body: What the heart does, how high BP and damaged arteries reduce blood flow to every system. Plain-language.
- WMTY: "Taking care of your heart can benefit your whole body."
- Stats: Nearly half of U.S. adults have high blood pressure.

**Kidney (K)**
- Headline: "Your kidneys do more than you think"
- Body: Kidneys filter waste, control blood pressure, manage fluid. When they struggle, the heart works harder.
- WMTY: "Monitoring your kidney health is a powerful step toward protecting your heart."
- Stats: About 1 in 7 U.S. adults have kidney disease.

**Metabolic (M)**
- Headline: "How your body uses energy matters"
- Body: Blood sugar, weight, fat storage. Conditions like diabetes and obesity create chronic stress on blood vessels.
- WMTY: "Small changes in how you eat, move, and sleep shape your CKM health over time."
- Stats: More than half of U.S. adults have prediabetes or diabetes.

**Heart + Kidney (H∩K)**
- Headline: "Heart and kidney problems often travel together"
- Body: Heart disease and kidney disease co-occur and accelerate each other. People with CKD are more likely to die from a heart attack than kidney failure.
- WMTY: "If you have one CKM-related health concern, screening all CKM components can catch problems early."

**Heart + Metabolic (H∩M)**
- Headline: "Diabetes and obesity put your heart at risk"
- Body: Elevated blood sugar damages arteries. Obesity raises blood pressure, promotes inflammation.
- WMTY: "Improving even one health factor improves the way your body functions."

**Kidney + Metabolic (K∩M)**
- Headline: "Blood sugar and blood pressure affect your kidneys"
- Body: Diabetes and high blood pressure are the two leading causes of chronic kidney disease worldwide.
- WMTY: "Keeping track of your blood glucose is one way to help protect your kidneys and overall health."

**Center — All Three (H∩K∩M)**
- Headline: "Everything is connected — and that's good news"
- Body: CKM syndrome is not a single disease; it's several connected conditions. But the connection means improvements in one area help your whole body. Life's Essential 8 gives you a proactive approach.
- WMTY: "CKM syndrome is preventable, treatable, and sometimes even reversible. You have more power than you think."
- Stats: Nearly 9 out of 10 U.S. adults age 20+ have at least one CKM risk factor.
- CTA: "Talk to your doctor about screening for CKM syndrome. Visit heart.org/myCKMhealth."

### 4.4 Content Sources

All consumer-facing copy must draw from these two approved documents:

1. **CKM Health Consumer Key Messages (Approved 6-3-2025)** — foundational narrative, primary/secondary messaging, calls to action
2. **CKM Health Consumer Messaging 2026 Update (Workgroup Review)** — updated definitions of CKM syndrome, risk factor stats, Life's Essential 8 framing, stage-specific CTAs

No statistics or health claims should be invented. Every number cited must trace back to one of these documents or the JAMA/AHA sources they reference.

## 5. Interaction & Animation Specifications

### 5.1 Diagram Animations

| Trigger | Animation | Duration |
|---------|-----------|----------|
| Page load | Circles fade in sequentially (H → K → M) | 300ms each, 150ms stagger |
| Hover on region | Scale to 1.03, brightness +10% | 200ms ease-out |
| Click region | Selected region to full opacity, others dim to 40% | 300ms ease |
| Panel open | Slide in from right (desktop) / up (mobile) | 350ms cubic-bezier |
| Panel close | Reverse of open | 250ms ease-in |
| Region visited | Checkmark fades in on label | 200ms |
| Nudge pulse | Unvisited regions glow softly | 2s infinite, begins after 3 visits |

### 5.2 Accessibility

- All regions have `role="button"`, `tabindex="0"`, and `aria-label` describing the region
- Keyboard navigation: Tab through regions, Enter/Space to select
- Focus ring visible on all interactive elements
- Content panel traps focus when open (modal behavior)
- All color contrasts meet WCAG 2.1 AA (4.5:1 minimum for text)
- Reduced-motion media query disables all animations
- Screen reader: region labels announce as "Heart health — click to learn more"

## 6. Analytics Plan

All tracking is anonymous — no account or PII required.

| Event | Data Captured |
|-------|---------------|
| `region_hover` | Region ID, timestamp |
| `region_click` | Region ID, timestamp, is_first_visit |
| `panel_view` | Region ID, duration visible |
| `panel_cta_click` | Region ID, CTA destination URL |
| `regions_explored` | Count of unique regions visited per session |
| `completion` | Fired when user visits all 7 regions |
| `return_visit` | Session-cookie check on page load |

**Key metrics aligned to project brief:**
- Completion rate (7/7 regions explored)
- Time spent learning (total panel-open duration)
- Topic interest patterns (which regions are explored first/most)
- Which formats drive understanding (click-through on CTAs)

## 7. Technical Approach

### 7.1 Component Architecture

The interactive will be built entirely within `src/components/ckmh-connections.vue` as a single-file Vue component, with content data pulled from `src/data/ckmh-connections.js`.

**Key internal structure:**

```
ckmh-connections.vue
├── <CkmhHero />          — Hero banner (already exists, will be adapted)
├── <VennDiagram />        — SVG-based interactive diagram
│   ├── <VennCircle />     — Individual circle (×3)
│   └── <VennRegion />     — Clickable overlap region (×7)
├── <ContentPanel />       — Slide-in detail view
│   ├── Heading + Body
│   ├── "Why This Matters" callout
│   ├── Key numbers
│   └── CTA link
└── <CompletionPrompt />   — Nudge after 4+ regions visited
```

These can be internal to the single `.vue` file or extracted into sub-components depending on complexity during build.

### 7.2 Data Model

The `ckmh-connections.js` file will be restructured from the current four-section array to a region-keyed map:

```js
export const ckmhData = {
  hero: { title, subtitle },
  regions: {
    heart:              { label, icon, headline, body, callout, stats, cta },
    kidney:             { label, icon, headline, body, callout, stats, cta },
    metabolic:          { label, icon, headline, body, callout, stats, cta },
    heart_kidney:       { label, icon, headline, body, callout, stats, cta },
    heart_metabolic:    { label, icon, headline, body, callout, stats, cta },
    kidney_metabolic:   { label, icon, headline, body, callout, stats, cta },
    center:             { label, icon, headline, body, callout, stats, cta },
  }
}
```

### 7.3 SVG Diagram Rendering

The Venn diagram will be rendered as an inline SVG within the Vue component. Each region is a `<path>` element calculated from circle intersections. This approach gives full control over hover states, transitions, and accessibility attributes without relying on canvas or external libraries.

### 7.4 Session Persistence

Use a lightweight cookie (no login) to store:
- Array of visited region IDs
- Visit count (for return-visit detection)
- Timestamp of last visit

This enables the "highlight unexplored regions" feature and return-visit analytics.

## 8. Open Questions & Decisions Needed

1. **Video/article embeds:** The project brief mentions "short videos and articles" in each section. Are there existing AHA video assets to embed, or is this text-only for v1?
2. **Life's Essential 8 integration:** The 2026 messaging doc emphasizes LE8. Should the center region link out to the LE8 tool, or should we embed a simplified version?
3. **CKM Staging:** The 2026 doc introduces CKM syndrome "stages." Should the interactive reference stages, or keep it simple for a general consumer audience?
4. **Sitecore integration:** The brief mentions Sitecore development. What are the integration requirements for embedding this Vue component within heart.org's Sitecore CMS?
5. **Mobile diagram treatment:** Should the Venn diagram maintain its overlapping form on mobile (with larger tap targets), or should it transform into a different layout (e.g., a list of tappable cards that still visually indicate connections)?

---

## 9. Success Criteria

The experience succeeds if:

- Users with zero health literacy can explain what CKM means after exploring the diagram
- Average session explores 4+ of 7 regions
- Completion rate (all 7 regions) exceeds 30%
- Content is read at a grade 5–8 level (validated by readability scoring)
- The experience meets WCAG 2.1 AA accessibility standards
- Users report the experience feels empowering, not overwhelming (qualitative testing)
